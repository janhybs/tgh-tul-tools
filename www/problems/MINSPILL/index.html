MINSPILL Optimize Vessel Spilling

You have N vessels with integer volumes V_i < 100 in litres. The biggest vessel is full of water, other vessels are empty. Your task is to measure W litres in the shortest time while the spilling of vessels has constant speed one litre per minute. One spilling either end with empty source vessel or full target vessel. Write a program that finds optimal sequance of spilling.

Input

On the first line, there is number of vessels N, on the following N lines there are volumes of the vessels. On the line N+2 there is number W of litres to measure.

Output

On the first line, there is number of spillings M. On following M lines, there are pairs of source and target vessel denoting individual spillings. Indices start from 0.

Example


Input:
3
7
5
3
1

Output:
3
0 2
2 1
0 2
Limits
Time limit
60s
Languages
All
