GOSTRVI Google Street View

When collecting data for the Google Street View service, the vehicle has to drive through every roud in the city. However, there are also one-way roads. Write a program that decides if the vehicle can drive through the city passing every road exactly once and return to the starting point. Multiple roads and loops are allowed.

Input

The first line contains two numbers, the number of crossroads N and number of roads M (or rather road directions). On following M lines, the roads are given by pairs of indices for the start and the target crossroad (indexed from 0). The two-way roads are given twice, once for every direction. One-way roads are given only in the direction they can be passed.

Output

On the output, there is the sequence of crossroad indices (one per line) on the found route. The sequence has to start and end in the crosroad 0 and where there are multiple possibilities you should output a sequence where roads with smaller indices are passed the first. If the closed route doesn't exist the output is empty.

Example


Input:
3 5
0 1
1 0
1 2
2 1
2 0

Output:
2
0
1
