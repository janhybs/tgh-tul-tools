TWOMISS Two missing from a sequence

The input is a set of numbers {0,...,n} with two numbers missing. Can you find them in linear time?

Input

First line contains number N, on the next N lines you get N numbers from the set {0,..., N+1}.

Output

Two missing numbers on single line. First the smaller one.

Example


Input:
5
6
1
3
5
0

Output:
2 4
Limits
Time limit
60s
Languages
All
