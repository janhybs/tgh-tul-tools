IDOS Route planer

Let us consider simplified system of public transport with N stations and a timetable that is same every day. One connection is given by the departure station, the time of departure (in minutes from the beginning of the day), the target station and time of journey (in minutes). Write a program that finds fastest route from station A to station B starting at time T. Assume that there are no connections over the midnight.

Input

The first line contains number of stations N and number of connections M. Following M lines contain data for individual connections: index of departure station, index of target station, time of departure, and time of journey. On the line M+2, there is number of test cases K that are given on following K lines. One test case is given by indices of stations A and B and the time T (in minutes from the beginning of the day).

Output

There will be K lines on the output, one for every test case, containing list of used conections in the fastest route. One connection in the list is given as the pair: index of the departure station and the time of departure encolosed in parenthesis. The last item of the list is index of the target station and time of arrival to the destination.

Example


Input:
4 6
0 1 10 30
0 1 70 30
0 2 130 10
1 2 60 30
1 3 100 60
3 1 30 20
2
0 2 0
0 3 20

Output:
(0 10) (1 60) (2 90)
(0 70) (1 100) (3 160)
